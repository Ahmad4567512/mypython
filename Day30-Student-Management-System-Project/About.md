# Day 30: Student Management System 

This is a simple **Student Management System** built using Python.  
It is a beginner-friendly console project to practice basic Python concepts.

## Features

- Add a student
- View all students
- Search a student by name
- Delete a student
- Simple menu-based system

## Concepts Used

- Lists
- Dictionaries
- Functions
- Loops (while, for)
- If-Else conditions
- String handling
- User input
